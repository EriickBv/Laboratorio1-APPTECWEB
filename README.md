## Descripción del Proyecto
Este sitio web es un portafolio informativo de una sola página que presenta los perfiles académicos y proyectos de los integrantes del equipo.

## Diseño Elegido
Hemos optado por un diseño de "tarjetas" independientes para cada sección, utilizando un fondo gris claro para resaltar los bloques de contenido blancos.
